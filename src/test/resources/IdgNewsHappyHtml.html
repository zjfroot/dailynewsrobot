Premiär för Amazons virtuella skrivbord

Daas-marknaden har begåvats med ytterligare en aktör. Amazon Workspaces körs på Windows Server 2008 R2 och ger en Windows 7-liknande upplevelse.

Amazon Web Services hostade virtuella skrivbord, så kallad daas (desktop as a service) är nu tillgängliga för allmänheten. Tjänsten går under namnet Workspaces och kostar från 230 kronor per användare och månad.

Till att börja med körs Workspaces i datacenter baserade i de amerikanska delstaterna North Virginia och Oregon, men inom kort kommer tjänsten att spridas till fler datacenter. Företag utanför USA är välkomna att utvärdera tjänsten, men om slutanvändarna befinner sig mer än 320 mil från något av datacentren får man räkna med sämre prestanda.



Skrivbordet som erbjuds är baserat på Windows Server 2008 R2 med RDS (Remote Desktop Service), något som ger användarna en Windows 7-liknande upplevelse.

Workspaces skrivbord kan integreras med Active Directory vilket gör att användarna kan logga in med sina vanliga uppgifter. I annat fall krävs inloggning med separat lösenord. När konfigureringen är klar kan användarna ansluta till sina skrivbord från mac, pc, Ipad eller enheter med Android. Stöd finns också för Amazons egen Kindle Fire-familj, men då behövs särskild mjukvara.



Det finns fyra olika paket att välja mellan. Standard och Standard Plus har en virtuell processor, 4,03 gigabyte minne och 50 gigabyte lagringsutrymme. Performance och Performance Plus har två virtuella processorer, 8,05 gigabyte minne och 100 gigabyte lagringsutrymme. Paketen kostar 230, 325, 390 respektive 490 kronor per användare och månad.

Standard och Performance erbjuder programmen Adobe Reader, Internet Explorer 9, Firefox, 7-Zip och Adobe Flash. Plus-versionerna inkluderar även Microsoft Office Professional 2010 och antivirusprogram från Trend Micro.

Amazon släpper nu även tjänsten Workspaces Sync som per automatik säkerhetskopierar användarnas dokument till S3.
